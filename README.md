# نخبة العقارية - Elite Properties

مشروع ويب متكامل لشركة عقارية متخصصة في بيع وشراء وتأجير العقارات. يقدم الموقع واجهة مستخدم عربية متكاملة لعرض العقارات المتاحة مع تفاصيلها وخدمات الشركة.

## المميزات الرئيسية

- واجهة مستخدم عربية متجاوبة تعمل على جميع الأجهزة
- عرض تفصيلي للعقارات المتاحة مع صور عالية الجودة
- أقسام متعددة تشمل: الرئيسية، العقارات المتاحة، من نحن، اتصل بنا
- تصميم عصري وأنيق مع تأثيرات حركية جذابة

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript
- Font Awesome للأيقونات
- خط Tajawal للغة العربية
- مكتبة AOS للحركات والانتقالات

## كيفية تشغيل المشروع

1. قم بتحميل الملفات
2. افتح ملف `index.html` في متصفح الويب المفضل لديك
